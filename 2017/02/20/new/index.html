2017年新的心跳

发表于 2017-02-20 | 分类于 总结
总结

前言

2016年这一年过得很快，但是足以发生许许多多的事情。如果要叙述一个复杂的故事，最简单的办法就是利用时间作为线索，采用顺序的叙述结构。这种结构同样非常适合个人学习总结。我毕业于2015年，本身对所学的机械设计行业并没有太大的兴趣,由于读的专业所以毕业后就进入了一家上市公司做机械设计，很遗憾一年时间工作还是没能提起兴趣所以决定年底辞职。2016年底开始筹划着转行,这段时间我边工作边开始了解互联网行业，通过知乎了解到互联网职位有产品经理、UI、前端、后端、运维等等这些职位的工作内容及需要技能。大学时期对互联网行业一直比较有兴趣，偏it的课程也就只有单片机C语言及PLC，选修过一门网页设计的课程知道有HTML和CSS。真正引我入前端是在腾讯工作的堂哥和在阿里巴巴工作的妹妹，通过他们介绍认识了2位在职的前端工程师，那时候才对前端知识体系及工作内容有了初步的认识，对前端所见所得及技术的迭代速度产生了浓厚的兴趣，并开始收集学习资料及制定了我为期2个月的学习计划。2016年12月20日辞职，开始自学前端。

上面所提到的虽不全面，但也大体覆盖了这一年发生的事情和所做的主要努力。下面的过程将开始分享自己过去两个月的前端学习历程。

过程

前端入门是一个螺旋上升的过程，既要反复看书看视频，也要必须抓紧时间实践练习。
我把这两个月大致分为四个阶段。(简述日后我会把记录的知识点详细发博客)

  • HTML + CSS

    • 理论:先读本书，毫无疑问入门是非常重要的，必须对知识有一个系统的认识，看完书后将知识点形成目录放在脑子里。HTML和CSS不是编辑语言，前者是结构语义化标签，后者则是样式配置。我是先看了一本书《HeadFirst HTML andCSS》，讲得通俗易懂，反复看了两遍。加入慕课网的前端工程师路径，迅速刷了两遍HTML和CSS基础教程。接下来就是深入学习CSS了,我是先看《CSS权威指南(第3版)》,了解了CSS的很多属性和特性，属性细节不比记忆，因为CSS本身就是需要结合其他标签用的,日后的项目用到肯定要再查的。重点放在对浮动、定位和盒模型这些重点认真看几遍。期间碰到问题都是浮动所带来的一些问题，我通过一些8小时学会HTML网页开发还有张鑫旭的博客弄清楚了，这些让我对CSS理解上升了一个层次。(虽然现在还有些不太理解，不过日后接触多定能想通)至此理论已经看得差不多了,很多知识点不懂我喜欢先反复看书看视频结合百度了解清楚。学完以上这些后，只要不是太复杂的页面，基本上能够临摹出差不多长相的了。
      自己做了2个简单的demo：导航条和个人简历。以上内容用时5天。
    • 实践: 下面开始是实践了，学习了几天HTML、CSS了，应该有点成果了。学习是必须有正反馈的，要知道经过自己研究努力做出来后知道自己哪里不足才有动力学，要觉得自己做出的东西有点用，才会继续学，所以我实践放在百度技术学院2016期,原因很简单因为项目开源，有其他人写的代码可以总结套路。刚开始写不太顺利，期间碰到很多问题，这是正常的，我是先根据不懂知识点查之前学过的w3c和CSS权威指南来复习，实在查不到才去看别人的代码，看一些开窍了，不断打代码，不断停顿，不断踩坑才有进步。刚开始打出第一张图的时候,代码出现了一些规范和顺序的问题，看完别人的代码后茅塞顿开。重新写几遍，开始重写发现很多属性和特性都理解了，写代码速度也快了很多。写到第三张的时候，基本上看到页面就知道怎么布局。由于时间原因我只做了一部分任务,但是着实让我的CSS得到了实践，好记性不如烂笔头，多敲多记才是最好的办法，总之ife真的很良心。后面看网络教程学会fireworks切雪碧图还有根据设计图还原了几个页面。
      这部分我做了2个项目:1.博雅首页。2.一个电商首页。(后续会将项目放在个人简历)。以上内容用时9天。
    • 进阶: 目前就到这个水平,进阶部分我准备学习:
      • 《精通CSS》汇集一些CSS最佳实践
      • 《响应式web设计》
      • 预处理工具Sass，自动化工具Gulp
      • 学会浏览器工具的使用
      • 常见的兼容性问题及解决方案的了解
      • 张鑫旭技术博客

  • JaveScript

    • 理论: 这部分是重点。后续很多框架都是这部分的衍生，故我很重视这部分的学习。刚开始是在网上了解了一些JavaScript的发展历史及演变，开始我是在w3c教程学习一阵子还是摸不着头脑，这部分需要的是涉及逻辑比较多，不像HTMLCSS像织毛衣一样，要学好对于初学者来说还是比较困难的。学习一门语言先从简单的概念开始引入一个新的概念来解释而不是引入更加复杂的概念。我先看了一遍《JavaScript DOM 编程艺术》，老老实实跟着敲，敲完后会发现很多效果都是网上的，发现自己也能做到，小有成就感，找到一点感觉，开始慢慢进入了编程的思维逻辑。接下来我是刷慕课网的JS教程慕课网JS视频，它是将JS的内容划分为JS基础，事件处理，JS动画及实践几个部分，这样会让你思路更清晰的去学习，而且每个知识点都有相应的代码练习，增强你对代码的理解和记忆。学习一门语言不是一蹴而就，上面部分只是初步有个感性的认知这门语言,想真正学会这门语言还需要不断的理解和记忆，任何一方面的缺失都是致命的。这些都是需要在实际工作项目中总结的，需要时间的。开始跟不上思路不要紧，对比其它编程语言，JS友好的弱类型语言好处就是学一点是一点。
      这部分看完，复习几遍。开始对浏览器的兼容性,和JS对页面对象的处理有一个初步了解。开始从无逻辑的代码到有逻辑的业务代码思想转变。以上用时14天。
    • 实践: 对JS语言有初步认识后，实践部分我主要是转到了网易云课堂JavaScript教程-从入门到精通,这部分我一共复习了三遍，JS一定要动手去打代码，看似简单的东西不去套路一遍，这个套路就不属于你。视频第一遍我是纯看视频理解每个知识点逻辑,这部分我是看视频没接触的概念查书《高级程序设计》(红宝书)和W3c教程进行消化理解的,期间不懂得概念我都是看一些比较零碎的视频(传智播客、妙趣课堂、饥人谷课件、慕课网、网易云课堂等等)或者百度找答案。代码演示可以在饥人谷的在线编辑器:JS Bin,方便操作和显示。然后第二遍复习是看视频跟着老师的讲解一步一步的写程序，学习完一遍以后，自知很多知识点掌握的还不是很牢固，然后果断作出了复习一遍，重写所有程序的决定。复习的时候，先是重新看一遍老师的讲解，巩固基础知识，然后将当节课的知识点尽量综合起来放在一个程序里面重写出来，这个过程就是举一反三的过程,。复习第三遍的时候,能自己根据业务需求用代码实现。原生JS这部分非常重要,如果没理解清楚,对后面的框架学习会很吃力,基础知识永远不会过时,课程其实很多基础知识没有涉及,下一步我将重点学习《高级程序设计》及其他书籍、blog来学习巩固基础知识。
      看完这部分,我做了2个demo:1. 轮播图。2.瀑布流。3.放大镜特效以上用时14天。
    • 进阶: 目前就到这个水平,进阶部分我准备学习:
      • 《高级程序设计》至少看三遍,基础知识很重要
      • 《JavaScript语言精髓》蝴蝶书
      • 《你不知道的JS》JS的坑一网打尽,闭包、this之类的问题找答案
      • 《ES6标准入门》ES6必须学会的内容，学会Webpack使用
      • 《JavaScript设计模式与开发实践》
      • 《高性能JavaScript》
      • 阮一峰的博客 学习JS

  • JQuery

    • JQuery这部分学起来会相对比较轻松,这部分是原生JS面向对象写出来的框架。jQuery作为JS的轻量级的库存在，自然而然的在应用层面上比原生JS有着很大的优势，在利用jQuery写效果的时候就能明显的感觉到这一点。在jQuery中各种选择器应有尽有，各种事件也非常多，还有很好用的链式写法，这一切都与jQuery的宗旨“Write less, do more”不谋而合。老方法,迅速慕课网JQuery教程,对基本用法有初步理解。无意中看到网易云李炎恢老师jQuery视频教程,感觉挺有意思就跟着学习完,对基本用法有初步记忆。这时候想起W3c教程,刷一遍进行总结,避免与JS语法混淆。由于时间计划安排这部分我通过w3c教程学习只掌握基本用法,代码未探究,下一步我将重点学习《锋利的JQuery》书本来进行学习与巩固,尝试探究JQuery代码。
      看完这部分,我做了2个demo:1.轮播图。2.自定义滚动条。以上用时14天。
    • 进阶: 目前就到这个水平,进阶部分我准备学习:
      • 《锋利的JQuery》学习

  • HTML5+CSS3+AJAX+Bootstrap

    • 这段时间由于过年节假氛围和来到别的城市需要租房和搬行李,刚开始浪费了一个星期时间在这里,不过还好我很快就调回状态了。这部分比较碎,HTML5、CSS3、AJXA看完传智博客2016年最新H5+CSS3教程视频,然后这部分一定要结合前面所学的知识进行复习和总结,了解新属性、智能表单、网页多媒体、SVG,重点在于HTML5新增的标签和CSS3的动画及标签。AJXA这部分重点学习AJAX的运行机制和原理,jsonp是怎么实现跨域的。看完后自己封装了一个简单的ajax,然后再应用此demo。Bootstrap看完慕课网玩转Bootstrap（基础）,看完后了解Bootstrap框架、使用Bootstrap制作Web页面或Web应用程序、能定制个性化Bootstra,看完后再看网易云Bootstrap视频教程。这部分比较琐碎,需要多看视频多百度来查不懂的概念,只能如此反复,多看几个人的答案,一个苹果交换后还是一个苹果,一种思想交换后就变成了两种思想。
      看完这部分,我做了1个项目:基于Bootstrap的PC/移动端响应式页面实战,以上用时14天。

展望

总结: 由于最近准备面试的简历和项目,自己开始将线下的日记和项目搬到github上。通过慕课网的学习版本控制入门 – 搬进 Github和版本管理工具介绍—Git篇申请了github的并且学会开源项目的基本操作与git的基本命令,并且用github+hexo搭建了这个博客。有兴趣请点击这篇文章手把手教你用Hexo+Github 搭建属于自己的博客,最后由于后续会将每个学习到的知识点放入博客的每篇文章以便于日后总结复习,我学习了慕课网在线代码编辑的方式,实现方法是将文章嵌入一个codeopen在线编辑器页面,其它页面可以放到github的gh-pages上以便于观看。
后面计划学习几种常见的框架Angular,Vue,脑补计算机基础知识:算法和数据结构。目前正在看华科课程,日后学习基础。很多知识刚开始接触比较困难,唯一的捷径就是多复习几次,所以有人说看一遍和看三遍《高级程序设计》是有巨大差别的,复习巩固才是真谛。手脑并用这一步也不能省略,大量模仿，无他，唯手熟尔。

后记: 至此,是我两个月自学前端的时间流。其实学的内容越多,就越感到自己知识面匮乏,工作实践项目接触少是我目前最缺乏的。大学期间我很多课程基本都是在图书馆自学总结并且能考出不错的成绩,这点从我的毕业绩点能看出。这段时间经历了过年的缓冲和转行的决心,给自己带上了一份特别的心理压力。很庆幸，这段时间以来自己能决心跨过传统机械行业,那种每天机械式工作的无力感转瞬被学习前端的兴趣所取代。两个月的自学,每天都充满斗志在学习和总结,也愈发觉得自己渐入佳境。

希望新的一年,家人及周围的亲戚朋友身体健康,一切顺利！也希望自己能找到一份满意的工作并实现自己的人生价值,做自己喜欢的事情,愿意为它加班的工作！

祝君好,一定要！
